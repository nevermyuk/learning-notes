## Learning Outcome

- Build infrastructure that delivers product and services at both speed and scale. 

- Continuous integration and continuous delivery (CI/CD) pipelines, 

- Deploy massive infrastructure securely using code.

- Operationalize microservices using Kubernetes.

### Project 3
